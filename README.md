# Turf Community Backend

TypeScript backend for the Turf Community platform using Supabase and Firebase Admin.

## Setup

1. Copy `.env.example` to `.env` and configure values.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```

## API Docs

Open `http://localhost:4000/docs` after starting the server.

## Built With

- Express
- Supabase
- Firebase Admin SDK
- Zod validation
- Pino logging
- Swagger UI
